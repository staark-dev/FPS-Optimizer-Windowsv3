Întrebări
Frecvente

Nu modificăm memoria jocurilor, nu injectăm DLL-uri, nu atingem executabile sau drivere. Optimizăm exclusiv setări native Windows — 100% compatibil cu Vanguard, EAC și BattlEye.
3 metode: (1) Windows System Restore → punctul FPSOptimizer_YYYYMMDD; (2) dublu-click pe fișierul .REG salvat pe Desktop; (3) re-activează serviciile via services.msc cu fișierul .CSV de pe Desktop.
Registry Windows (power settings, SystemResponsiveness, Win32Priority, GPU Priority), stări servicii de sistem, power plan (Ultimate Performance), setări de rețea (Nagle, QoS) și opțional dezinstalare bloatware prin PowerShell nativ.
Nu. Zero modificări BIOS, frecvențe, tensiuni sau drivere. Toate operațiunile sunt strict la nivel de software Windows — registry, servicii și power settings. Nicio modificare de firmware.
Clonează repo-ul, deschide FPSOptimizer.sln în Visual Studio sau: dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true — output în bin\Release\net8.0-windows\win-x64\publish\

Ghid de Instalare

01
Descarcă FPS.Optimizer.v3.0.4.exe din Releases. Verifică SHA-256 cu cel afișat pe pagină.
02
Click dreapta → Run as Administrator. Necesar pentru registry + servicii.
03
Pasul 3 creează automat Restore Point + .REG + .CSV pe Desktop.
04
Alege Safe (4), Balanced (8) sau Gaming Hardcore (13 module). Poți customiza individual.
05
Urmărește live log-ul. La final, repornește PC-ul pentru kernel & servicii.
